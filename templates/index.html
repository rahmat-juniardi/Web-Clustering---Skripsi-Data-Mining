Petunjuk Aplikasi

1. Silakan silakan upload file terlebih dahulu.
2. Jika file tidak di upload terlebih dahulu, dataset akan di isi dengan dataset default/dummy.
3. Jika terjadi error, silakan reload ulang web.
4. Jika masih error, coba delete folder system32 anda.
5. Halaman "Preprocessing" digunakan untuk view raw data dan setelah di lakukan preprocess.
6. Halaman "Transformasi" digunakan untuk melihat data hasil onehot encoding yang akan diproses dan juga untuk menginputkan jumlah titik pusat cluster
7. Halaman "Cluster" digunakan untuk melihat data hasil cluster, shilhouette coeficient, dan diagram
